# Sample Exports

This folder contains export samples generated by the Python ADT application so we can compare them side by side with the legacy ADT outputs.

Default sample case:
- 4 faces per level
- panel offset `0.340 m`
- first panel heading `0.0 deg`
- 4 levels
- vertical spacing `1.150 m`
- alternate levels cogged `false`

Generator:
- [`scripts/generate_sample_exports.py`](../scripts/generate_sample_exports.py)

Default output folder:
- `samples_exports/default_case_4face_4level`

To regenerate:

```powershell
.\venv\Scripts\python.exe .\scripts\generate_sample_exports.py
```
